Dokumentasi offline tailwind css

  • Ahmad Habibi
  • May 31, 2022

Tailwindcss adalah framework css yang banyak di pakai oleh para web developer karena fleksibilitas nya. Maka dari itu saya juga tidak mau ketinggalan untuk memakai framework keren ini. Kendalanya saat saya mau ngoding dan internet tidak ada, dan lupa class class dari tailwind, karena tidak membuka dokumentasi nya, dan tentunya itu cukup sulit. 1. Download file-file web nya

Buka terminal, lalu clone dengan git, atau juga bisa di download manual dari github 

git clone https://github.com/tailwindlabs/tailwindcss.com.git

Kemudian masuk ke direktori file yang kita clone tadi

cd ./tailwindcss.com

2. Install

Oh ya, website tailwindcss.com ini di buat menggunakan framework JavaScript bernama Nextjs . Jadi, kita perlu menambah paket-paket yang diperlukan untuk menjalankan website nya.

Yang kita butuhkan pertama adalah yarn, sebuah package manager untuk Node.js . Jika belum di install silahkan diinstall. Kalau anda tidak memakai linux, silahkan cari sendiri cara menginstall Node.js dan Yarn

# Untuk Arch
sudo pacman -S yarn

#Untuk Debian
sudo apt install yarn

#Untuk Fedora
sudo dnf install yarn

Install dependencies yang di butuhkan

yarn

Jika sudah, maka website tailwindcss sudah bisa dijalankan

yarn dev

Dan… hasil nya adalah seperti website tailwindcss.com
